< Temperature Model >

This app retrieves the daily temperature forecast for a user-specified location from Open-Meteo Weather Forecast API. Then, it constructs a query to feed into Image-Charts API, which returns a line graph for users to visualize the temperature. For fun, I added functionality with a fun fact API, so next to the average daily temperature is a fun fact about that number.
